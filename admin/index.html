Meta Pixel (Facebook / Instagram Ads)

El píxel de Meta permite rastrear conversiones y crear audiencias para tus campañas en Facebook e Instagram. Encuéntralo en: Meta Business Suite → Events Manager → Crear píxel.

Google Analytics 4

GA4 te permite ver el tráfico, comportamiento de usuarios y conversiones de tu sitio. Encuéntralo en: analytics.google.com → Crear propiedad.

Google Tag Manager

GTM permite gestionar todos tus píxeles y scripts sin tocar el código. Recomendado si vas a manejar múltiples herramientas.
